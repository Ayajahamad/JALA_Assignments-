CSS Positioning Examples

Static Positioning

Static

Static positioning is the default. Elements are positioned according to the normal document flow.

Relative Positioning

Relative

Relative positioning moves an element relative to its normal position, without affecting the layout of surrounding elements.

Fixed Positioning

Fixed

Fixed positioning keeps an element fixed to the viewport, meaning it will stay in the same position even when the page is scrolled.

Absolute Positioning

Absolute

Absolute positioning removes an element from the normal document flow and positions it relative to the nearest positioned ancestor.

Sticky Positioning

Sticky

Sticky positioning toggles between relative and fixed positioning, depending on the user's scroll position. The element sticks to the top of the viewport when you reach its scroll position.
