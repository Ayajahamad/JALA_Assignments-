Handling Overflow Using CSS

The CSS overflow property controls what happens to content that overflows an element's box. Here we explore different values of the overflow property.

Overflow: Visible (Default)

This box has overflow: visible. Overflowing content is not clipped and will render outside the box.

With overflow: visible;, the overflowing content is not clipped and will render outside the element's box. This is the default value.

Overflow: Hidden

This box has overflow: hidden. Overflowing content is clipped, and no scrollbars are visible.

With overflow: hidden;, the overflowing content is clipped, and the user will not be able to see the overflowed content.

Overflow: Scroll

This box has overflow: scroll. Scrollbars are always visible, even if the content does not overflow.

With overflow: scroll;, the element will display scrollbars regardless of whether the content overflows or not. This forces the user to scroll to see the overflowed content.

Overflow: Auto

This box has overflow: auto. Scrollbars are added only if the content overflows.

With overflow: auto;, the element will only display scrollbars if the content overflows. This is useful for making scrollbars appear only when necessary.

Overflow-X and Overflow-Y

This box has overflow-x: scroll and overflow-y: hidden. Horizontal scrollbar is visible, but vertical content is clipped.

You can control the overflow behavior of the horizontal and vertical axes independently using overflow-x and overflow-y.
