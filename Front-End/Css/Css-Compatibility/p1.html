Table Layout Example

The table-layout property defines the layout algorithm used to arrange cells, rows, and columns in a table. The two common values are fixed and auto.

With table-layout: fixed, the table and column widths are set by the widths of the table and the widths of the columns, regardless of the content inside the cells. It allows for a more predictable layout.

With table-layout: auto (the default), the column widths are determined by the content inside the cells. The table will adjust the width of columns based on the content, making it more flexible but less predictable.
